Analisis Sentimen | EDOM Politeknik Harapan Bersama

Website ini merupakan sistem yang dapat menganalisa komentar mahasiswa terhadap penilaian subjektif kepada dosen kedalam 4 kategori yaitu kurang, cukup, baik dan sangat baik!

Dataset

8092 Dataset berupa komentar mahasiswa terhadap evalusi dosen digunakan sebagai data latih dengan pembagian 80% sebagai data latih dan 20% data testing. presentasi kategori isi data : 28.7% (Kurang), 21.1% (Cukup), 15.3% (Baik) dan 28.9% (Sangat Baik)

Extrasi Fitur

Term frequency-inverse document frequency (TF-IDF) digunakan untuk pembobotan kata pada dataset komentar mahasiswa sebagai bentuk numerik untuk pemodelan pada aplikasi website ini.

Modeling

Long short term memory network (LSTM) digunakan sebagai algoritma modeling yang menghasilkan akurasi sangat baik dengan memanfaatkan library earlystoping sehingga mendapatkan akurasi terbaik dalam modeling.

Library

Library yang digunakan untuk membangun model menggunakan numpy, pandas, nltk, tensorflow dan sastrawi serta beberapa library lain.
